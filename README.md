# Area Calculator — Java Swing GUI

<div align="center">
  <img src="https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white" />
</div>


This feature allows you calculate the area of common 2D shapes (like circles, rectangles, triangles, etc.) with automatic unit conversions. 

---

### Supported Shapes:
- Circle
- Rectangle
- Triangle
- Square
- Trapezoid
- Octagon
- Parallelogram
- Ellipse 
- Sector 

---

### Universally Supported Units:
- `m`, `cm`, `mm`, `dm`, `km`
- `in`, `ft`, `yd`, `mi`

---

### 🛠 Tech Stack:
- **Language:** Java
- **GUI Framework:** Swing
- **Platform:** independent
